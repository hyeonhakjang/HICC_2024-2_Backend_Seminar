# 2024-2 HICC 백엔드 세미나

2024년도 2학기 HICC 백엔드 세미나 자료입니다. 4주차에 걸쳐 다음의 세미나를 진행합니다.

해당 세미나는 python 언어에 대한 세미나가 아닙니다. 기초적인 python 문법 지식이 있는 분만 수강 바랍니다.
